# Whitefalls Livery

Static one-page luxury chauffeur website for Whitefalls Livery, built with plain HTML, CSS, and JavaScript.

## Files

- `index.html` - semantic page structure and original marketing copy
- `style.css` - premium black-and-gold responsive design system
- `script.js` - mobile navigation, sticky header state, and reveal/smooth-scroll interactions

## Customization

- Replace placeholder contact details in `index.html` before launch.
- Update testimonials once real client feedback is available.
- Adjust colors in `:root` inside `style.css` if the brand palette evolves.

## Usage

Open `index.html` in a browser. No build step or dependencies are required.
